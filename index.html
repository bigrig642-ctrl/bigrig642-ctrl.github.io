Current Setup Flow

Primary Setup Options

One-page setup manual for the current Discord bot systems. Use the setup commands below to configure welcome messages, vouches, roles, tickets, applications, channels, and the full config view.

Core Setup 7 Slash Commands

/setupwelcome

Configure the welcome channel, welcome message text, enable state, preview mode, and embed toggle.

/setupvouch

Set the vouch channel, enable or disable the system, and toggle embed mode for posted vouches.

/setuproles

Assign Staff, Helper, PM, Ticket Staff, and Staff Apps Review roles with reset support.

/setuptickets

Build up to 7 ticket types with labels, categories, ping roles, viewer roles, allowed opener roles, and button style.

/setupapps

Create application types, review channels, questions, reviewer roles, and role grants on acceptance.

/setupchannels

Choose utility channels such as vouches, partner tracking, ticket logs, task deadlines, and giveaways.

/setupview

Open a full config summary showing enabled systems, saved channels, roles, panels, and missing-reference warnings.
