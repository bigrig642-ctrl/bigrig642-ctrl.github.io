Primary Setup Options

Main configuration commands

Start here when setting up a new guild or refreshing the server configuration.

/setupwelcome

Configure the welcome channel, welcome message text, enable state, preview mode, and embed toggle.

/setupvouch

Set the vouch channel, enable or disable the system, toggle embed mode, and choose whether vouch messages are allowed.

/setuproles

Assign Staff, Helper, PM, Ticket Staff, and Staff Apps Review roles with reset support.

/setuptickets

Build up to 7 ticket types with categories, ping roles, viewer roles, allowed opener roles, button style, and panel publishing.

/setupapps

Create application types, review channels, questions, reviewer roles, and role grants on acceptance.

/setupchannels

Choose utility channels such as vouches, partner tracking, ticket logs, task deadlines, giveaways, and pricing hooks.

/setupview

Open a full config summary showing enabled systems, saved channels, roles, panels, stored vouches, and warnings.
